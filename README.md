## Orange 开发框架是一款 Golang 语言 web 开发框架

### 主要特性
- 基于 golang 性能不凡；
- MVC 分层设计，逻辑清晰；
- 核心功能短小精悍，灵活度高；
- 丰富的工具包，有 图片验证码，本地缓存，发送邮件，http请求客户端，日志 等常用工具包；
- 持续更新，已应用于多个线上成熟项目；
- 源码地址 [https://gitee.com/zhucheer/orange](https://gitee.com/zhucheer/orange)
